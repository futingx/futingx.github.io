# 使用 uv 配置 12306 MCP Server 完整指南

> 本文记录在 Windows 环境下，通过 pip 安装 uv 并配置 12306 MCP Server 的详细步骤。

---





## 环境说明

- **操作系统**：Windows 11
- **Python 版本**：3.10+
- **目标应用**：Chatbox（支持 MCP 的 AI 客户端）

---

## 第一步：安装 uv

我们使用 pip 方式安装 uv，我尝试了官方的几种方法，这是最简单快捷的方法。

### 1.1 打开终端

按下 `Win + R`，输入 `cmd`，回车打开命令提示符。

### 1.2 执行安装命令

```bash
pip install uv
```

### 1.3 验证安装

```bash
uv --version
```

如果显示类似以下内容，说明安装成功：

```
uv 0.11.1 (a6042f67f 2026-03-24 x86_64-pc-windows-msvc)
```

---

## 第二步：配置 Chatbox

### 2.1 打开 Chatbox 设置

1. 打开 Chatbox 应用
2. 进入 **设置** → **MCP 服务器**
3. 点击 **添加 MCP 服务器**

### 2.2 填写配置信息

| 字段 | 值 |
|------|-----|
| **名称** | `12306` |
| **类型** | `本地 (stdio)` |
| **命令** | `uvx` |
| **参数** | `mcp-server-12306` |
| **环境变量** | （留空） |

### 2.3 完整 JSON 配置

如果需要手动编辑配置文件，添加以下内容：

```json
{
  "mcpServers": {
    "12306": {
      "command": "uvx",
      "args": ["mcp-server-12306"]
    }
  }
}
```

> 💡 **提示**：`uvx` 是 uv 的工具运行器，可以直接运行远程 Python 包，无需手动下载和安装项目。

---

## 第三步：重启并测试

1. **重启 Chatbox** 使配置生效
2. 在聊天窗口中输入查询指令，例如：
   - `查询车次 G222 的途径站信息`
   - `北京到上海的高铁有哪些`

---

## 常见问题

### Q1: uvx 命令找不到

确保 uv 已正确安装，重新打开终端试试：

```bash
uv --version
```

### Q2: Chatbox 无法连接 MCP

确认 JSON 格式正确，重启 Chatbox 检查日志。

---

## 总结

使用 uv 配置 MCP 非常简便，整个流程只需：

1. `pip install uv` → 安装 uv
2. 配置 Chatbox（或者其他例如cherrystdio这样的客户端） → 添加 MCP 服务器（使用 uvx）
3. 重启测试

# 使用 uv 配置高德地图 MCP Server 完整指南

> 本文记录在 Windows 环境下，通过 uv 配置高德地图 MCP Server 的详细步骤。

---

## 环境说明

- **操作系统**：Windows 11
- **Python 版本**：3.10+
- **目标应用**：Chatbox（支持 MCP 的 AI 客户端）
- **前置条件**：已安装 uv（参见上篇文章）

---

## 第一步：申请高德地图 API Key

### 1.1 注册/登录高德开放平台

1. 访问 [高德开放平台](https://console.amap.com/)
2. 使用账号登录（如没有则注册）
3. 进行个人实名认证（必须）

### 1.2 创建应用

1. 进入 **应用管理** → **创建应用**
2. 填写应用名称，例如：`Chatbox MCP`
3. 应用类型选择：**Web 服务**

### 1.3 添加 Key

1. 进入应用详情页
2. 点击 **添加 Key**
3. 填写 **Key 名称**
4. 勾选 **Web服务**（必须）
5. 点击提交，获取 Key

> ⚠️ **注意**：新申请的 Key 可能需要 20 分钟左右生效

---

## 第二步：配置 Chatbox

### 2.1 打开 Chatbox 设置

1. 打开 Chatbox 应用
2. 进入 **设置** → **MCP 服务器**
3. 点击 **添加 MCP 服务器**

### 2.2 填写配置信息

| 字段 | 值 |
|------|-----|
| **名称** | `amap` |
| **类型** | `本地 (stdio)` |
| **命令** | `uvx` |
| **参数** | `amap-mcp-server` |
| **环境变量** | `AMAP_KEY=你的API Key` |

### 2.3 完整 JSON 配置

如果需要手动编辑配置文件，添加以下内容：

```json
{
  "mcpServers": {
    "amap": {
      "command": "uvx",
      "args": ["amap-mcp-server"],
      "env": {
        "AMAP_KEY": "你的高德API Key"
      }
    }
  }
}
